# Simple Downloads Folder Organiser

A simple Python script that automatically organises files in my Downloads folder
into categorised directories based on their file extensions.

## How It Works

The script scans my `Downloads` directory and moves files into these folders:

* **Images**: jpg, jpeg, png, gif, webp
* **Documents**: doc, docx, txt
* **PDFs**: pdf
* **Spreadsheets**: xls, xlsx, csv
* **Archive**: zip
* **Videos**: mp4, mov
* **Software**: dmg
* **Config**: json, yml, yaml
* **Other**: Any file types not listed above

## Requirements

* Python 3.4 or higher
